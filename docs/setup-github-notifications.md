# GitHub Notifications

In order to receive GitHub email notifications, you may have to update your account settings on GitHub.  Navigate to `Settings -> Notifications` in your account and make the changes listed below.
<br>
<br>
![alt text](./images/Notifications.png "notifications")<br>  

## Update Watching Notifications
The default is just GitHub notifications. Make sure you also select Email as an option.<br>
<br>
![alt text](./images/Watching.png "watching")<br>  

## Participating mentions and custom
The default is just GitHub notifications. Make sure you also select Email as an option.<br>
<br>
![alt text](./images/Participating.png "participating")<br>

## Email Updates
There are no options selected by default. Select the options you want here.<br>
<br>
![alt text](./images/EmailUpdates.png "email updates")<br>
